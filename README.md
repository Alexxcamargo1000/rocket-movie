# RocketMovies

Uma aplicação para anotar seus filmes.

<img src="https://i.imgur.com/t7iTMLe.png" alt="imagem do site" width="50%"/>


## Instalação

Instale RocketMovie com npm

```bash
  npm install
```

Para rodar o projeto 

```bash
  npm run dev
```
